的复制构造函数和复制操作
当一个对象生成或者赋值的时候，会从构造函数里面找有没有符合参数的工作函数，然后复制或者调用赋值构造函数


1.
显示对用构造函数
string a("hetong")


2.
显示调用构造构造函数构造临时对象，调用复制构造函数复制对象到a
string a = string("hetong")

3.
void func(const string a)
func("hetong")

调用string构造函数生成临时对象，调用复制构造函数

4.
void func(const string &a)
func("hetong")
调用string构造函数生成临时对象，然后引用对象

5.
对象复制
string v1;
string v2;
v1 = v2;


6.
string v1;
v1 = “hetong”;

调用构造函数，然后对象复制
