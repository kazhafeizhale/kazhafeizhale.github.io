平衡字符串
题目描述：
在一个「平衡字符串」中，'L' 和 'R' 字符的数量是相同的。

给出一个平衡字符串 s，请你将它分割成尽可能多的平衡字符串。

返回可以通过分割得到的平衡字符串的最大数量。


---------------------------------------------------------------------------------------------------------------------------------
示例 1：

输入：s = "RLRRLLRLRL"
输出：4
解释：s 可以分割为 "RL", "RRLL", "RL", "RL", 每个子字符串中都包含相同数量的 'L' 和 'R'。

int balancedStringSplit(char * s){
    int num = 0;
    int j = 0;
    for(int i=0;i<strlen(s);i++)
    {
        if(s[i] == 'R')
        {
            j++;
        }
        else
        {
            j--;
        }
        if(j == 0)
        {
            num++;
        }
    }
    return num;
}
